# Laundry Wallah

Laundry Wallah is a simple static website for a laundry and dry-cleaning service. It showcases the business services, pricing, booking form, and contact details in a clean and user-friendly layout.

## Features
- Welcome section for the brand
- Service listing for washing, ironing, dry cleaning, and more
- Price table for common services
- Booking form for customer inquiries
- Contact section with email support

## Files
- index.html: Main homepage content and layout
- OIP.jpg: Brand image used on the homepage

## How to View
Open index.html in your browser to view the website locally.
